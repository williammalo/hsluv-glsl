# HSLUV-GLSL

HSLUV is a human-friendly alternative to HSL. ( http://www.hsluv.org )
GLSL port by William Malo ( https://github.com/williammalo )
Put this code in your fragment shader.

## api

	  xyzToRgb (vec3 xyz   )
	  rgbToXyz (vec3 rgb   )
	  xyzToLuv (vec3 xyz   )
	  luvToXyz (vec3 luv   )
